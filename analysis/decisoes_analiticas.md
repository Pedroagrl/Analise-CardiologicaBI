# Decisões Analíticas do Projeto

Este projeto foi desenvolvido como parte de um trabalho freelance para uma clínica, com foco na aplicação de Análise de Dados e Business Intelligence no contexto da saúde, utilizando dados clínicos estruturados para analisar o risco cardíaco e a relação entre sintomas clínicos e exames diagnósticos.

---

## Separação dos Dashboards

O projeto foi dividido em dois dashboards distintos:

1. Triagem e Perfil de Risco Cardíaco  
2. Exames Clínicos e Confirmação Diagnóstica  

Essa separação foi adotada para evitar sobrecarga cognitiva e manter uma narrativa analítica clara, diferenciando sintomas clínicos de exames diagnósticos.

---

## Variáveis Não Utilizadas nos Dashboards

Algumas variáveis presentes no conjunto de dados original, como colesterol, açúcar alto e batimentos cardíacos máximos, dentre outras variáveis foram mantidas no dataset bruto, porém não utilizadas nos dashboards desenvolvidos nesta etapa.

A decisão foi tomada por essas variáveis não contribuírem diretamente para os objetivos analíticos definidos.
